General information on data processing

This website processes personal data only to the extent necessary for providing a functional website and its contents. Processing happens on the legal basis of Art. 6 (1) (f) GDPR (legitimate interests).

Server log data

When you visit this site, the hosting provider automatically collects technical data (IP address, browser type, operating system, referring URL, time of request). This data is stored temporarily and is not merged with other data sources.

External resources

This variant of the site loads fonts from Google Fonts and the CodeMirror 6 editor library from the esm.sh CDN. When these resources are loaded, your IP address is transmitted to the respective provider. No personal data beyond this technical request is collected by the site operator.

Cookies

This site does not use tracking cookies. The live playground keeps editor state only in your browser's current tab and never transmits code you type to any server.
